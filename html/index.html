About Me

Hello I'm Andrew Lyon, and I welcome to my site. I appreciate you taking the time out of your day to visit. In February of 2018 I decided to switch careers and was accepted into Skill Distillery, a sixteen-week intensive coding boot camp to become a Java Full-Stack Developer. In June 2018 I graduated from Skill Distillery and also married the love of my life, Perla. I am currently a Software Developer I at NexGen Technologies Inc., a company located right outside of Denver, CO.

Although I am new to this career field, I have a B.A. in History and an Associate’s in Cyber Security. I also have an extensive history of teaching others. I have gained experience in full stack application development including Java programming, object-oriented development, RESTful services, Spring MVC, Angular, and SQL. Our program strongly emphasized pair-programming.

Check Out Some of My Projects

Blackjack

This application simulates the popular casino game, Blackjack. I completed this project as a homework assignment after my third week at Skill Distillery. The game allows you to place bets add additional players that are computer simulated that will also place bets. Black Jack was my first project of this size and it brought together everything we had learned at Skill Distillery so far including conditionals, array lists, enums, and object-oriented programming.

Blackjack was a very fun project for me to create. It is run from the console and requires you to have java installed to compile and run. If you have the resources and time, it would well be worth you checking out!

Unfortunate Super Heroes

Unfortunate Super Heroes is a creative Create, Read, Update, Delete (CRUD) app that allows you to perform the four basic CRUD functions on a list of superheroes that unfortunately aren't that super. This project was the result of my week 8 homework at Skill Distillery. Unfortunate Super Heroes was my first JPA based website. I also created the (very basic) MySQL schema as well as adding some filler data.

Feel free to mess around with this application and add your own superhero, don't make him too fortunate!

Todo List

Todo List was my first angular project. It was built in my last few weeks at Skill Distillery. The application allows you to add things to a todo list, edit them, and check them off as complete. This version does not have the register/login functions implemented, but I will be uploading that version shortly.

I invite you to mess around with the app and add your own todos to the list.
